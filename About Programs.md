## 🚀 How to Run the Programs

1. Install Python (if not installed)
2. Clone this repository:

```bash
git clone https://github.com/adity303/Python-Programs-.git
```

3. Navigate to the folder:

```bash
cd Python-Programs-
```

4. Run any file:

```bash
python filename.py
```
